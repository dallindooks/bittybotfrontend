This chart contains the data of all trades made from the bot's creation date to today. A trade is considered Unprofitable if BTC is bought at a certain price and sold at a price lower than it was bought.

This chart aggregates all of the trades in a single day and displays the result visually. This will allow the user to find which days were most profitable.

This chart helps determine the relationship between prediction certainty in a price increase vs how profitable a trade was. The R^2 value is based on a linear regression of the current values.
